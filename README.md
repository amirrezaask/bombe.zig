# cpu.zig
cpu is a cpu simulator.

## Spec
- 16 * 64 bit registers
- Stack based memory
- heap memory

## Operations
** Each Ref can be either a register or a const **
- SET REF1 Ref2 
- ADD OUTREF REF1 REF2
- SUB OUTREF REF1 REF2
- MUL OUTREF REF1 REF2
- DIV OUTREF REF1 REF2
- POW OUTREF REF1 REF2
- ROOT OUTREF REF1 REF2
